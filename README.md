# Wi-Fi Probe Request Sniffer

## Description
This Python project uses Scapy to detect nearby devices sending Wi-Fi probe requests. It prints unique Wi-Fi network names that devices are searching for.

## Features
- Detects SSIDs of nearby networks
- Filters duplicates
- Displays output in real-time
- Logs network names (optional)

## Tech Used
- Python 3
- Scapy
- Linux (monitor mode required)

## Sample Output
[+] Detected New Probe Request: SolaX_wireless
[+] Detected New Probe Request: globalnET/JMEL
